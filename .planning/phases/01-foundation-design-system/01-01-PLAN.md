    **Step 1: Build the 3D Grid Layout System (`src/components/layout/grid-system.tsx`).**

    This is NOT a standard CSS grid. It's a spatial composition system that gives sections a 3D depth feel -- like looking at a dashboard of floating panels at slightly different z-depths. This is what makes ULTRASTREAM look like abyssinecator.com, not a generic landing page.

    Create two components: `GridSystem` (the container) and `GridCell` (individual cells).

    `GridSystem` props:
    - `children: React.ReactNode`
    - `columns?: number` (default 12 -- standard grid)
    - `className?: string`

    Implementation:
    ```tsx
    'use client'

    import { cn } from '@/lib/utils/cn'

    interface GridSystemProps {
      children: React.ReactNode
      columns?: number
      className?: string
    }

    export function GridSystem({ children, columns = 12, className }: GridSystemProps) {
      return (
        <div
          className={cn(
            // Base grid
            'grid gap-4 md:gap-6 lg:gap-8',
            // 3D perspective container -- this creates the spatial depth
            '[perspective:1200px]',
            // Responsive columns
            'grid-cols-1 md:grid-cols-6 lg:grid-cols-12',
            // Padding
            'px-4 md:px-8 lg:px-12 xl:px-16',
            // Max width centered
            'max-w-[1440px] mx-auto',
            className
          )}
          style={{
            gridTemplateColumns: undefined, // Let Tailwind handle responsive
          }}
        >
          {children}
        </div>
      )
    }
    ```

    `GridCell` props:
    - `children: React.ReactNode`
    - `colSpan?: { sm?: number; md?: number; lg?: number }` (responsive column spans)
    - `depth?: 'near' | 'mid' | 'far'` (z-depth for 3D effect)
    - `className?: string`
    - `glow?: 'blue' | 'purple' | 'gold' | 'none'` (optional neon glow border)

    Implementation:
    - The `depth` prop maps to CSS transforms: `near` = `translateZ(20px)`, `mid` = `translateZ(0px)`, `far` = `translateZ(-20px)` with corresponding scale and opacity adjustments
    - Use `transform-style: preserve-3d` on cells
    - The `glow` prop adds a subtle box-shadow using the theme glow colors
    - On mobile (sm), all cells are full-width and depth effects are disabled (flat layout)
    - Transition hover: cells slightly lift on hover (`translateZ(30px)` for near, etc.)

    ```tsx
    interface GridCellProps {
      children: React.ReactNode
      colSpan?: { sm?: number; md?: number; lg?: number }
      depth?: 'near' | 'mid' | 'far'
      glow?: 'blue' | 'purple' | 'gold' | 'none'
      className?: string
    }

    const depthStyles = {
      near: 'lg:[transform:translateZ(20px)] lg:scale-[1.02]',
      mid: 'lg:[transform:translateZ(0px)]',
      far: 'lg:[transform:translateZ(-20px)] lg:scale-[0.98] lg:opacity-90',
    }

    const glowStyles = {
      blue: 'shadow-[var(--shadow-neon-blue)]',
      purple: 'shadow-[var(--shadow-neon-purple)]',
      gold: 'shadow-[var(--shadow-gold)]',
      none: '',
    }

    export function GridCell({
      children,
      colSpan = { sm: 1, md: 6, lg: 6 },
      depth = 'mid',
      glow = 'none',
      className,
    }: GridCellProps) {
      return (
        <div
          className={cn(
            // Base cell styling
            'rounded-lg border border-white/5 bg-obsidian/50 backdrop-blur-sm',
            'p-6 transition-all duration-500',
            '[transform-style:preserve-3d]',
            // Depth effect (only on lg+)
            depthStyles[depth],
            // Glow effect
            glowStyles[glow],
            // Hover lift
            'hover:lg:[transform:translateZ(30px)] hover:lg:scale-[1.03]',
            // Responsive column spans
            `col-span-${colSpan.sm || 1}`,
            `md:col-span-${colSpan.md || 6}`,
            `lg:col-span-${colSpan.lg || 6}`,
            className
          )}
        >
          {children}
        </div>
      )
    }
    ```

    IMPORTANT: The col-span classes must be safelisted or use inline styles since Tailwind cannot detect dynamic class names. Use a `style` prop for gridColumn instead:
    ```tsx
    style={{
      gridColumn: `span ${colSpan.lg || 6}`,
    }}
    ```
    And use Tailwind's responsive prefix approach with media queries in the style, OR use a mapping object with pre-defined Tailwind classes. The cleanest approach: use `style` prop with CSS custom properties and media queries handled by the parent grid's responsive behavior. Test this carefully.

    **Step 2: Build the Epic Preloader (`src/components/layout/preloader.tsx`).**

    This is the first thing visitors see. It sets the tone for the entire site: dark, cinematic, powerful.

    This MUST be a client component ('use client') because it uses state and animations.

    Implementation:
    - Full-screen overlay fixed to viewport, z-index 9999, bg-void (#050505)
    - Centered ULTRASTREAM text logo (using display font, large, with a metallic gold gradient text effect)
    - Below the logo: a minimal loading bar or pulse animation in neon-blue
    - Animation sequence using CSS animations (NOT GSAP -- keep this lightweight for initial load):
      1. Logo fades in with a slight scale-up (0 -> 300ms)
      2. Loading bar animates from 0% to 100% width (300ms -> 1500ms)
      3. Entire preloader slides up and fades out (1500ms -> 2000ms)
    - Use `useState` to track loading state. After the animation completes (~2s), set `isLoaded` to true.
    - When `isLoaded` is true, remove the preloader from the DOM (or set display:none) to prevent it blocking interactions.
    - Use `onAnimationEnd` or `setTimeout` for timing.
    - Add a CSS class `preloader-done` to the body when complete so other components can trigger entrance animations.

    ```tsx
    'use client'

    import { useState, useEffect } from 'react'
    import { cn } from '@/lib/utils/cn'

    export function Preloader() {
      const [phase, setPhase] = useState<'loading' | 'exiting' | 'done'>('loading')

      useEffect(() => {
        // Loading phase
        const exitTimer = setTimeout(() => setPhase('exiting'), 1800)
        const doneTimer = setTimeout(() => {
          setPhase('done')
          document.body.classList.add('preloader-done')
        }, 2300)

        return () => {
          clearTimeout(exitTimer)
          clearTimeout(doneTimer)
        }
      }, [])

      if (phase === 'done') return null

      return (
        <div
          className={cn(
            'fixed inset-0 z-[var(--z-preloader)] flex flex-col items-center justify-center',
            'bg-void transition-all duration-500',
            phase === 'exiting' && 'opacity-0 -translate-y-8'
          )}
        >
          {/* Logo */}
          <h1
            className={cn(
              'text-5xl md:text-7xl font-display font-bold tracking-wider',
              'bg-gradient-to-r from-gold-dark via-gold to-gold-light bg-clip-text text-transparent',
              'animate-[fadeInScale_0.6s_ease-out_forwards]'
            )}
          >
            ULTRASTREAM
          </h1>

          {/* Loading bar */}
          <div className="mt-8 w-48 h-[2px] bg-white/10 rounded-full overflow-hidden">
            <div
              className="h-full bg-gradient-to-r from-neon-blue via-neon-purple to-neon-blue rounded-full animate-[loadBar_1.5s_0.3s_ease-in-out_forwards]"
              style={{ width: '0%' }}
            />
          </div>

          {/* Tagline */}
          <p className="mt-4 text-text-muted text-sm tracking-[0.3em] uppercase animate-[fadeIn_0.5s_0.8s_ease-out_forwards] opacity-0">
            The Revolution Loads
          </p>
        </div>
      )
    }
    ```

    Add the keyframe animations to `globals.css`:
    ```css
    @keyframes fadeInScale {
      from {
        opacity: 0;
        transform: scale(0.9);
      }
      to {
        opacity: 1;
        transform: scale(1);
      }
    }

    @keyframes loadBar {
      from { width: 0%; }
      to { width: 100%; }
    }

    @keyframes fadeIn {
      from { opacity: 0; }
      to { opacity: 1; }
    }
    ```
