# ULTRASTREAM

## What This Is

ULTRASTREAM is a decentralized, high-performance streaming and short-form video platform — a Web3-native competitor to Twitch, YouTube, TikTok, and Discord, specializing in gaming content. It uses NEAR Protocol for identity and finance, a P2P Mesh Network for video delivery, and the KZR (Kruzer) token economy for creator monetization and viewer rewards. ULTRASTREAM is a companion to ULTRAVERSE.games (the parent platform) but stands as its own independent product.

**Current scope: Marketing/landing website** — a hype page with waitlist, feature showcase, and community-building to drive signups before the full platform goes live. The platform itself is being built by a separate team.

## Core Value

Creators earn more, viewers get rewarded, and no single authority can deplatform anyone — a streaming platform that puts financial sovereignty and content freedom back in the hands of its community.

## Context

**Parent Ecosystem:** ULTRAVERSE.games is the parent platform. ULTRASTREAM is both a companion and standalone product. Streamers get unique discovery opportunities with new videos boosted beyond existing channels.

**Competitive Landscape:** Direct competitors are Twitch (50% revenue cut, poor discoverability), YouTube (algorithm-driven, ad-heavy), TikTok (short-form, censorship concerns), Discord (community but no streaming), Kick (newer but still centralized). ULTRASTREAM aims to kill all pain points of these platforms.

**Technical Stack (Platform — for reference):**
- NEAR Protocol — identity (Named Accounts like creator.ultrastream), governance (SputnikDAO), smart contracts
- libp2p — P2P peer discovery and mesh streaming
- Arweave/Filecoin/IPFS — decentralized storage for stream archives
- Livepeer — decentralized transcoding network
- RTMP/SRT — ingest protocols (OBS, vMix, Streamlabs compatible)
- KZR Token — Kruzer cryptocurrency for tipping, subscriptions, engagement rewards

**Key Differentiators:**
- **Creator Monetization:** 0-5% platform fee vs Twitch's 50% cut. Smart subscriptions via smart contracts.
- **Viewer Rewards:** Proof-of-Engagement faucet distributes KZR for verified watch-time. Bandwidth contributors rewarded.
- **Discovery Engine:** "Social-Proof" algorithm based on KZR transactions and community sentiment, not just watch-time. New creators get boosted.
- **Censorship Resistance:** P2P mesh delivery, decentralized storage, DAO governance (The Architects council).
- **Privacy-Respecting Age Verification:** Age ID system that verifies without collecting personal data — scrambles anything private.
- **Meaningful Subscriptions:** Not just a badge — recurring payments managed by smart contracts with real value.

**Governance:** The Architects — an independent council using SputnikDAO on NEAR for platform policy, featured content, and dispute resolution.
